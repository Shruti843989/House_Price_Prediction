# House Price Prediction using Linear Regression

This project predicts house prices using Machine Learning.  
A Linear Regression model is used to predict the selling price of houses based on different features.

## Steps
- Data loading
- Data cleaning
- Encoding categorical data
- Train-test split
- Model training using Linear Regression
- Model evaluation

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Target
SalePrice (House Price Prediction)
